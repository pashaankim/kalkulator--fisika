# ⚛️ Smart Physics Calculator

Aplikasi kalkulator fisika cerdas berbasis Streamlit dengan fitur:
- 🧮 Smart Calculator (18 rumus fisika + langkah pengerjaan)
- 📚 Formula Library dengan LaTeX
- 🔄 Auto Unit Converter (9 kategori)
- 📊 Interactive Graph (Matplotlib)
- 🤖 AI Physics Assistant (rule-based parser)
- 🔬 Virtual Lab (simulasi jatuh bebas, gelombang, pegas)
- 📝 Practice Questions (8 soal + pembahasan)
- 🔢 Scientific Calculator
- 📋 Physics Cheat Sheet
